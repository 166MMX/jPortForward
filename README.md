# jPortForward

## Features
* Multiple Listening ports
* Multiple Targets per Listening port
* Filter remote access
** IP address and/or port
** CIDR notation and/or port

## Dependencies
* Spring Framework
* jCommander
* Logback
** JCL over SLF4J (to get Spring talk to Logback)
** Java mail
** Groovy
** Janino
